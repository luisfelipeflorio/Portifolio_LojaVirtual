## Visão Geral do Projeto

E-commerce B2C de confeitaria artesanal desenvolvido com Python/Django full stack e TailwindCSS.

- **Stack:** Python 3.12 · Django 5.x · SQLite · TailwindCSS 3.x
- **Interface:** Responsiva (mobile-first), inteiramente em português brasileiro
- **Código:** PEP 8, aspas simples, em inglês, CBVs preferencialmente

Referência completa de requisitos: [`PRD.md`](../PRD.md)
